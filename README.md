## ローカル開発

```bash
# リポジトリをクローン
git clone https://github.com/[username]/fundamental-math-with-claude.git
cd fundamental-math-with-claude

# ローカルサーバーで確認（例：Python）
python -m http.server 8000

# ブラウザで http://localhost:8000 を開く
```

## GitHub Pages へのデプロイ

1. GitHubでリポジトリを作成
2. main ブランチにプッシュ
3. Settings → Pages → Source を "main" ブランチに設定
4. 数分後にサイトが公開される
