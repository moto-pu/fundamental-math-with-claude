位相の例

集合と位相 第3章

位相の定義は抽象的ですが、具体例を通じて「何を表しているのか」を掴んでいきましょう。

例1：離散位相（最も細かい位相）

定義：離散位相

集合 $X$ のすべての部分集合を開集合とする位相です。

$$ \mathcal{O} = \mathcal{P}(X) = \{A \mid A \subset X\} $$

具体例：$X = \{a, b, c\}$ の場合

開集合の族 $\mathcal{O}$ は：

𝒪 = {
  ∅,
  {a}, {b}, {c},
  {a,b}, {a,c}, {b,c},
  {a,b,c}
}

全部で $2^3 = 8$ 個の開集合があります。

公理の確認

(O1) ∅ ∈ 𝒪 ✓,  X = {a,b,c} ∈ 𝒪 ✓
(O2) 例：{a,b} ∩ {b,c} = {b} ∈ 𝒪 ✓
(O3) 例：{a} ∪ {c} = {a,c} ∈ 𝒪 ✓

すべての部分集合が開集合なので、どんな演算をしても結果は必ず $\mathcal{O}$ に入ります。

直感的な意味

離散位相では、すべての点が他の点から「孤立」しています。 

点 a の最小の開近傍 = {a}（a だけを含む）

各点を「完全に分離」できるので、これは最も「細かい」区別ができる位相です。

💻 プログラマ向け比喩
// 離散位相 ≈ すべてのオブジェクトが完全に区別可能
// equals() が厳密な参照比較のようなもの

if (obj1 == obj2)  // 同一オブジェクトのみ true

例2：密着位相（最も粗い位相）

定義：密着位相

開集合が $\emptyset$ と $X$ だけの位相です。

$$ \mathcal{O} = \{\emptyset, X\} $$

具体例：$X = \{a, b, c\}$ の場合

𝒪 = {∅, {a,b,c}}

たった2個の開集合しかありません。

公理の確認

(O1) ∅ ∈ 𝒪 ✓,  X ∈ 𝒪 ✓
(O2) ∅ ∩ X = ∅ ∈ 𝒪 ✓,  X ∩ X = X ∈ 𝒪 ✓
(O3) ∅ ∪ X = X ∈ 𝒪 ✓

直感的な意味

密着位相では、どの点も他の点と「区別できない」状態です。 

点 a を含む最小の開集合 = X 全体（a だけを取り出せない！）

すべての点が「くっついて」いて分離できません。

💻 プログラマ向け比喩
// 密着位相 ≈ すべてのオブジェクトが等価
// equals() が常に true を返すようなもの

if (obj1.equals(obj2))  // 常に true

例3：$\mathbb{R}$ の標準位相（ユークリッド位相）

これが最も重要で、解析学で使う位相です。

定義

開区間 $(a, b) = \{x \in \mathbb{R} \mid a < x < b\}$ を基本として、 開区間の任意の和集合を開集合とします。

開集合の例

✓ 開集合である：
  (0, 1)           -- 開区間
  (0, 1) ∪ (2, 3)  -- 開区間の和集合
  (-∞, 5)          -- 無限開区間
  ℝ 全体           -- 全体集合
  ∅                -- 空集合

✗ 開集合でない：
  [0, 1]           -- 閉区間（端点を含む）
  [0, 1)           -- 半開区間
  {0}              -- 一点集合
  ℤ（整数全体）     -- 離散的な点の集まり

なぜ $[0, 1]$ は開集合でないのか？

開集合の定義を思い出すと：

点 $x$ が開集合 $O$ に属する ⟺ $x$ を中心とする十分小さな開区間が $O$ に含まれる

端点 0 を考えてみましょう：

0 を含む開区間 (0-ε, 0+ε) = (-ε, ε)

これは負の数を含むので [0,1] に収まらない！

つまり、端点 0 の「周り」を $[0, 1]$ 内に取れないので、$[0, 1]$ は開集合ではありません。

視覚的イメージ

開集合 (0, 1)： ──────(●━━━━━━━━●)────── 0 1 ↑ 端点を含まない（開いている） 閉集合 [0, 1]： ──────[●━━━━━━━━●]────── 0 1 ↑ 端点を含む（閉じている） 半開区間 [0, 1)： ──────[●━━━━━━━━●)────── 0 1 開でも閉でもない

位相の「粗さ」について

同じ集合 $X$ 上でも、異なる位相を入れられます。

密着位相 ⊂ ... ⊂ 標準位相 ⊂ ... ⊂ 離散位相
（粗い）                              （細かい）
  • 位相が細かい = 開集合が多い = 点をより細かく区別できる
  • 位相が粗い = 開集合が少ない = 点の区別があいまい

比較表

位相 開集合の数 点の分離 特徴
離散位相 $2^n$（最大） 完全に分離 すべてが見分けられる
密着位相 2（最小） 分離不能 何も区別できない
$\mathbb{R}$の標準位相 無限 分離可能 解析学の基礎
有限補位相 無限 一点は分離不能 代数幾何で使用

演習問題

問題1

$X = \{1, 2, 3\}$ 上で、以下は位相になるか？

  1. $\mathcal{O} = \{\emptyset, \{1\}, \{1,2,3\}\}$
  2. $\mathcal{O} = \{\emptyset, \{1\}, \{2\}, \{1,2,3\}\}$
  3. $\mathcal{O} = \{\emptyset, \{1\}, \{2,3\}, \{1,2,3\}\}$
問題2

$\mathbb{R}$ の標準位相で、$\mathbb{Q}$（有理数全体）は開集合か？閉集合か？

解答

問題1

(a) $\mathcal{O} = \{\emptyset, \{1\}, \{1,2,3\}\}$

(O1) ∅ ∈ 𝒪 ✓, {1,2,3} ∈ 𝒪 ✓
(O2) {1} ∩ {1,2,3} = {1} ∈ 𝒪 ✓
(O3) {1} ∪ {1,2,3} = {1,2,3} ∈ 𝒪 ✓

答え：位相である ✓

(b) $\mathcal{O} = \{\emptyset, \{1\}, \{2\}, \{1,2,3\}\}$

(O3) {1} ∪ {2} = {1,2} ∈ 𝒪 ???

→ {1,2} は 𝒪 に含まれていない！ ✗

答え：位相ではない ✗

(c) $\mathcal{O} = \{\emptyset, \{1\}, \{2,3\}, \{1,2,3\}\}$

(O1) ∅ ∈ 𝒪 ✓, {1,2,3} ∈ 𝒪 ✓
(O2) {1} ∩ {2,3} = ∅ ∈ 𝒪 ✓
(O3) {1} ∪ {2,3} = {1,2,3} ∈ 𝒪 ✓

答え：位相である ✓（分割位相）

問題2

$\mathbb{Q}$ は開集合でも閉集合でもない

有理数と無理数はどちらも $\mathbb{R}$ の中で稠密であるため、 どの有理数の周りにも無理数があり、どの無理数の周りにも有理数があります。 したがって、$\mathbb{Q}$ もその補集合も開集合ではありません。
