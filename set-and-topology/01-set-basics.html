集合論の基礎

集合と位相 第1章

集合とは

集合とは、ある条件を満たす「もの」の集まりです。 集まりを構成する個々のものを元（げん）または要素と呼びます。

例：基本的な集合
A = {1, 2, 3, 4, 5}      -- 有限集合
ℕ = {0, 1, 2, 3, ...}    -- 自然数全体（無限集合）
ℤ = {..., -2, -1, 0, 1, 2, ...}  -- 整数全体
ℚ = 有理数全体
ℝ = 実数全体

元 $x$ が集合 $A$ に属することを $x \in A$ と書き、 属さないことを $x \notin A$ と書きます。

集合の記法

外延的記法

要素を直接列挙する方法です。

A = {2, 4, 6, 8, 10}

内包的記法

条件を使って定義する方法です。

A = {x ∈ ℕ | x は偶数かつ x ≤ 10}

これは「自然数 $x$ のうち、偶数かつ10以下のもの全体」を表します。

部分集合

定義：部分集合

集合 $A$ のすべての元が集合 $B$ にも属するとき、 $A$ は $B$ の部分集合であるといい、$A \subset B$ または $A \subseteq B$ と書きます。 

{1, 2} ⊂ {1, 2, 3}
ℕ ⊂ ℤ ⊂ ℚ ⊂ ℝ

任意の集合 $A$ に対して、$\emptyset \subset A$（空集合はすべての集合の部分集合）であり、 $A \subset A$（自分自身は部分集合）です。

集合演算

演算 記号 定義
和集合 $A \cup B$ $\{x \mid x \in A \text{ または } x \in B\}$
共通部分 $A \cap B$ $\{x \mid x \in A \text{ かつ } x \in B\}$
差集合 $A \setminus B$ $\{x \mid x \in A \text{ かつ } x \notin B\}$
補集合 $A^c$ 全体集合 $X$ に対して $X \setminus A$
直積 $A \times B$ $\{(a, b) \mid a \in A, b \in B\}$

ド・モルガンの法則

定理：ド・モルガンの法則
$$ (A \cup B)^c = A^c \cap B^c $$ $$ (A \cap B)^c = A^c \cup B^c $$

これはプログラミングにおける論理演算と同じ構造です。

// プログラミングでの対応
!(a || b) == (!a && !b)
!(a && b) == (!a || !b)

写像

定義：写像

集合 $X$ から集合 $Y$ への写像 $f$ は、 $X$ の各元 $x$ に対して $Y$ の元 $y$ をただ一つ対応させる規則です。

$$ f: X \to Y, \quad x \mapsto f(x) $$
  • 定義域：$X$
  • 値域：$f(X) = \{f(x) \mid x \in X\}$
  • 終域：$Y$

写像の種類

単射（injection）

異なる元は異なる像に移ります。

$$ x_1 \neq x_2 \Rightarrow f(x_1) \neq f(x_2) $$
全射（surjection）

すべての $y \in Y$ に対して $f(x) = y$ となる $x$ が存在します。

$$ f(X) = Y $$
全単射（bijection）

単射かつ全射。逆写像が存在します。

全単射があるとき、二つの集合は「同じ大きさ」（同じ濃度）を持つと考えます。

族について

「族（ぞく）」は集合論でよく出てくる用語で、 直感的には「集合の集まり」のことです。 

A = {1, 2, 3}                      ← 元の集まり（集合）
ℱ = {{1}, {2,3}, {1,2,3}}          ← 集合の集まり（族）

添字付けられた族

添字集合を使って集合を並べたものも族と呼びます。

{Aλ}λ∈Λ  または  (Aλ)λ∈Λ

例：
  Λ = {1, 2, 3}
  A₁ = {a, b}
  A₂ = {c}
  A₃ = {a, c, d}
  
  族 {Aλ}λ∈Λ = {{a,b}, {c}, {a,c,d}}
💻 プログラマ向けの比喩
# 添字付き族 ≈ 辞書（Dictionary / Map）
family = {
    1: {'a', 'b'},
    2: {'c'},
    3: {'a', 'c', 'd'}
}

# Λ = family.keys() = {1, 2, 3}
# A₂ = family[2] = {'c'}
